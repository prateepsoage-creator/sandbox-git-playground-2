# กะบะทราย 2 : ละครโครงการ (Sandbox Project)

Repository นี้เป็นโครงการจำลอง  
ใช้สำหรับฝึกการใช้งาน Git และ GitHub  
ในมุมมองของงานวิศวกรรม / EPC / งานโครงการจริง

## วัตถุประสงค์
- ฝึกคิด Git เป็นแฟ้มงานทั้งโครงการ
- ฝึกใช้ commit แทนเหตุการณ์สำคัญของงาน
- ฝึกดูประวัติการแก้งาน (history) แบบมีเรื่องราว
- ใช้เป็นสนามฝึกก่อนนำไปใช้กับงานจริงของ SOAGE

## ขอบเขต
- ไม่ใช่งานจริง
- ทดลอง แก้ ลบ ได้ทั้งหมด
- เน้นความเข้าใจ ไม่เน้นความสวยงาม

## แนวคิดหลัก
- 1 repository = 1 โครงการ
- 1 commit = 1 เหตุการณ์ของโครงการ
- history = ไทม์ไลน์ของงาน
